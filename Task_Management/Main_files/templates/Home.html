Task Management Dashboard

The Task Management App is a simple and efficient platform designed to help users manage their daily tasks effectively. It allows users to:

  • View Tasks: See all tasks in a well-organized list with their current status (completed or pending).
  • Add Tasks: Easily add new tasks through a user-friendly form.
  • Update Status: Mark tasks as completed with a single click.
  • Delete Tasks: Remove tasks that are no longer needed (optional).
